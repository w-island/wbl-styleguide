# wbL Design System

wbL(더블유아일랜드 주얼리 브랜드) 디자인 시스템 스타일 가이드입니다.
세이지 그린 · 아이보리 · 차콜을 중심으로 한 에디토리얼 미니멀 무드의 토큰·컴포넌트·가이드라인을 담고 있습니다.

## 보기

- **라이브:** https://w-island.github.io/wbl-styleguide/
- 단일 HTML 파일(`index.html`)이며 별도 빌드 과정이 없습니다. 파일을 브라우저에서 바로 열어도 됩니다.

Button · Badge & Label · Form & Input · Tab & Chip · Alert & Toast · Photography · UX Writing · Do / Don't
